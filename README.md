# 🎮 OW2 Helper

Overwatch 2プレイヤーのための総合ヘルパーアプリ

[日本語](README.md) | [English](README_EN.md)

## ✨ 機能

### 🗺️ マップコールアウト辞書
- 25+マップの場所の呼び方を収録
- 日本語・英語両対応
- 検索機能付き

### ⚔️ カウンターピック支援
- 全41ヒーロー対応
- 敵チームを選択 → おすすめカウンター提案

### 📊 メタ分析ダッシュボード
- 8ランク帯対応（ブロンズ〜GM+）
- ロール別ヒーローランキング
- 勝率・ピック率表示

### 📅 イベントカウントダウン
- 次のイベント・アップデートまでをリアルタイム表示

## 🚀 使い方

1. このリポジトリをクローン
```bash
git clone https://github.com/yourusername/ow2-helper.git
```

2. `index.html` をブラウザで開く
